## 背景與目的

台北市／新北市「生生喝鮮乳」政策讓持有悠遊卡的幼兒可至超商兌換鮮乳。若家長忘記帶實體卡，官方允許以「識別碼 + 16碼卡號」手動輸入兌換。本網站提供：

1. 輸入兩組號碼後自動生成一維條碼，讓超商掃碼取代手動輸入
2. 儲存多筆悠遊卡資料於 localStorage，可重複叫用
3. 引導找出正確欄位（識別碼位置、卡號位置）
4. 兒童優惠卡（愛心卡）無識別碼時，引導至官方平台查詢

**主要使用情境：** 家長在超商，手機操作，時間緊迫。

---

## 技術架構

### 技術選型

- **單一 `index.html` 檔案**（純 Vanilla JS + 內嵌 CSS）
- 條碼函式庫：[JsBarcode](https://github.com/lindell/JsBarcode)（CDN 載入）
- 資料儲存：`localStorage`（key: `easycard_profiles`）
- 部署：GitHub Pages，直接 push `index.html` 即可上線
- 零後端、零 build 步驟、完全離線可用

### 條碼規格

- 格式：**Code 128**（超商條碼掃描器通用）
- 兩組條碼分開顯示：**識別碼／學號**（較小）+ **16碼卡號**（較大）
- 若掃碼失敗，畫面同時顯示兩組號碼供手動輸入

## 畫面設計

### 視覺風格

藍白政府感：主色 `#1a56db`，背景 `#f0f4ff`，白色卡片，乾淨可信任感。手機直向優先。字體：標題 ≥16px，內文 ≥14px。

### 畫面 1：首頁（卡片列表）

- 標題：「🥛 悠遊卡換鮮乳條碼產生器」
- 已儲存的悠遊卡以卡片列表顯示，每張卡片顯示：
  - 悠遊卡暱稱（大字）
  - 「顯示條碼」按鈕
  - 點擊整張卡片或按鈕均可進入條碼頁
- 底部「＋ 新增悠遊卡」虛線按鈕
- 長按卡片 → 顯示「編輯」「刪除」選項
- 無資料時顯示空狀態提示，引導新增第一筆

### 畫面 2：新增 / 編輯表單

欄位：

| 欄位 | 說明 |
|------|------|
| 悠遊卡暱稱 | 文字輸入，用於識別用 |
| 識別碼／學號（正面）| 文字輸入 |
| 16碼卡號（背面）| 數字輸入，輸入時自動每4碼插入空格（儲存時去除） |

**兒童優惠卡識別碼提示（永遠顯示於說明區塊下方）：**

> ⚠️ 兒童優惠卡正面無識別碼，請先至官方平台查詢：
> [前往悠遊卡多元兌換平台查詢識別碼 →](https://mepweb.easycard.com.tw/portal/about)
> （查詢時選縣市＋行政區，輸入16碼卡號，即可找到識別碼）

**欄位說明（可展開）：**

> **如何找到這兩個號碼？**
> - 識別碼/學號：卡片**正面**右上角或左下角（幼兒數位卡在左下方）
> - 16碼卡號：卡片**背面**中央印刷的數字

底部按鈕：「儲存並生成條碼」

### 畫面 3：條碼顯示

- 返回按鈕（左上）
- 悠遊卡暱稱（大字，藍色）
- **識別碼／學號 Code 128 條碼**（較窄）+ 識別碼明碼

- 黃色提示框：「⚠️ 如掃碼失敗，請店員手動輸入：識別碼 [xxx] + 卡號 [xxxxxxxxxxxxxxxx]」
- 底部小字：「請截圖保存此頁面」
- 底部操作：「✏️ 編輯資料」「🗑 刪除」

## 資料模型

```json
// localStorage key: "easycard_profiles"
// 值為 JSON 陣列
[
  {
    "id": "uuid-v4",
    "name": "小明",
    "type": "general",
    "identifier": "A12345678",
    "cardNumber": "1234567890123456",
    "createdAt": "2026-04-21T00:00:00Z"
  }
]
```

- `name`：悠遊卡暱稱，用於區分多張卡
- `type`：預設 `"general"`，保留欄位供未來擴充
- `cardNumber` 儲存時去除空格，存純數字字串
- UUID 用 `crypto.randomUUID()` 生成

## 錯誤處理與驗證

| 情境 | 處理方式 |
|------|---------|
| 16碼卡號非16位數 | 儲存前驗證，顯示紅色提示文字 |
| 識別碼為空 | 顯示提示：「請填入識別碼（或先至官網查詢）」 |
| 悠遊卡暱稱為空 | 無法儲存，顯示提示 |
| localStorage 不可用 | 顯示警告：「此瀏覽器不支援資料儲存，資料將在關閉頁面後消失」，仍可生成條碼 |
| JsBarcode CDN 載入失敗 | 顯示錯誤訊息：「條碼無法生成，請確認網路連線」 |

## 不在範圍內
